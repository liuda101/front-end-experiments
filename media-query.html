Media Query

Media Query on a link element. 

<link rel="stylesheet" media="print" href="some/path/to/css.css" />

Media Query within a style sheet. 

@media(print){/* some css rules */}

Media types:

all
所有设备
braille
盲文设备
embossed
handheld
手持设备
print
打印设备，或者处于打印浏览模式的设备
projection
投影设备
screen
电脑屏幕
speech
语言合成设备
tty
tv
电视

Logical operators : not, and, only 

@media screen (min-width:700px) and (orientation: landscape){ }
@media (min-width :700px), handheld and (orientation: landscape){ }

Media features: color color-index aspect-ratio device-aspect-ratio device-height device-width grid height monochrome orientation resolution scan width
